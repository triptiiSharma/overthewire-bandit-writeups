# 🚩 OverTheWire: Bandit — Writeups & Walkthrough

A structured, level-by-level documentation of the OverTheWire: Bandit wargame. This repository serves as a personal reference and a practical demonstration of foundational Linux and command-line security concepts explored through hands-on challenges.

## About OverTheWire: Bandit

[OverTheWire](https://overthewire.org/wargames/) is a platform that offers wargames designed to help individuals learn and practice security concepts in a legal, controlled environment. **Bandit** is the entry-level wargame and is recommended as the starting point for absolute beginners.

Each level of Bandit introduces progressively complex challenges centered around Linux file navigation, permissions, networking, and scripting — all of which form the foundation of real-world cybersecurity work.

## Repository Structure
```
OverTheWire-Bandit/
├── README.md
└── levels/
    ├── level0.md
    ├── level0-1.md
    ├── level1-2.md
    ├── level2-3.md
    ├── level3-4.md
    ├── level4-5.md
    ├── level5-6.md
    ├── level6-7.md
    └── ...
```

Each file inside the `levels/` directory corresponds to a single level transition and follows a consistent structure:

- **Level Goal** — the objective as stated by the challenge
- **Commands Used** — all commands involved in solving the level
- **Walkthrough** — step-by-step breakdown of the solution
- **Concepts Covered** — key Linux/security concepts introduced or reinforced

---

## How to Use This Repository

1. Visit [OverTheWire: Bandit](https://overthewire.org/wargames/bandit/) and start at Level 0.
2. Attempt each level independently before referring to any writeup.
3. Navigate to the corresponding file in the `levels/` directory if you need guidance.
4. Use the concepts covered section to reinforce your understanding beyond just the solution.

> This repository is intended as a learning aid, not a shortcut. Working through the challenges yourself before consulting the writeups will yield the most value.
